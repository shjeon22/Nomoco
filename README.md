## 1. 어떤 프로젝트 인가? 
2019년 말부터 현재까지 중국에서 최초 보고되고 퍼지기 시작한 코로나 바이러를 예방하기 위해 코로나 바이러스의 정보공유알림을 위한 홈페이지 입니다.

&nbsp;

## 2-1. 주요 기능
1.회원가입/로그인
2.관리자(회원관리)
3.게시판(페이징,답글)
4.코로나 현황 크롤링
5.선별진료소 위치정보(Kakao Map API)


&nbsp;

## 2-2. 개발 환경 / 기술 스택은?
**개발환경**

- STS 
- Apache Tomcat (8.5)
- Mybatis
- maven

**기술 스택**

- Java (jdk 1.8.0)
- spring Framework 
- MySQL DB (8.0.29)
- javascript
- jQuery
- HTML5
- CSS

&nbsp;
